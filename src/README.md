# Java Learning Journey ☕

Este repositório documenta minha jornada de aprendizado em desenvolvimento Java.

Os projetos foram criados ao longo de uma trilha de estudos focada em backend, abordando desde fundamentos de Programação Orientada a Objetos até o desenvolvimento de APIs com Spring Boot.

Cada projeto representa um módulo ou conjunto de conceitos aprendidos durante o curso.

---

## Tecnologias e conceitos estudados

- Java
- Programação Orientada a Objetos
- Collections
- Estrutura de dados
- JDBC
- Spring Framework
- Spring Boot
- APIs REST
- Banco de Dados
