# Sistema de Pedidos

Este projeto foi desenvolvido para praticar conceitos de Programação Orientada a Objetos em Java.

## Objetivo

Simular o funcionamento básico de um sistema de pedidos utilizado em lojas, permitindo cadastro de produtos, criação de pedidos e cálculo do valor total.

## Conceitos aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Listas (ArrayList)
- Ordenação de listas
- Organização de projeto em pacotes

## Tecnologias utilizadas

- Java
- IntelliJ / Eclipse
- Git

## Aprendizados

Durante o desenvolvimento deste projeto foi possível praticar a modelagem de entidades e aplicar conceitos fundamentais da programação orientada a objetos.
